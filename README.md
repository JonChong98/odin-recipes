# odin-recipes

This project is part of the Odin Project foundations course on HTML/CSS.

This starter project will make use of the knowledge learnt so far and give
a chance to apply said knowledge in creating a website. Don't be too 
concerned about the look of the website yet since CSS hasn't been covered
much yet, just focus on the html side things and familiarizing with git.

Also it's been a while since I did any HTML stuff so this will be good
for practice and a refresher.
